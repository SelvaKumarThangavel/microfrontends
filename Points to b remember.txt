Command for Angular application:
1. ng new angular_App_name - to create a new application
2. ng g c component_name - to create a component
3. ng g c component_name -it -is - to create component with in-line template and styles
4. ng g s service_name - to create a service
5. ng serve or npm start - to start the angular application
6. npm run serve - to run the express and sequelize
7. npm install express body-parser cors sequelize
8. to access via IP address run the application with ng serve --host 0.0.0.0 command


Command for react application:
1. npm start - to start react application

Micro front Ends:
